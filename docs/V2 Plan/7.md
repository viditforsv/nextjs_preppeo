# Product Roadmap - Adaptive Learning LMS

## Vision

> "Teach one concept at a time, in the simplest possible way, based on data, not assumption."

---

## Phase 1: Foundation (Months 1)

**Goal:** Basic adaptive system with manual rules

### Core Features

- **IBDP Mathematics Focus** - All 4 courses (AA/AI HL/SL) with 2000+ questions
- **Course Structure** - Units → Chapters → Topics → Lessons → Tags
- **Question Bank** - Tagged questions with difficulty levels
- **Student Progress** - Track accuracy, time, attempts per question
- **Basic Adaptation** - RYG rules (Red=reteach, Yellow=practice, Green=advance)
- **Admin Dashboard** - View student progress and tag mastery

### Technical Setup

- Supabase database with core tables
- Next.js frontend with question cards
- Basic analytics dashboard
- Automated question tagging system + Manual QC

---

## Phase 2: Intelligence

**Goal:** Smart recommendations and retention tracking

### New Features

- **Retention Engine** - Track what students forget over time
- **Stamina Detection** - Identify when students get tired
- **Smart Recommendations** - Suggest next questions based on weak tags
- **Spaced Repetition** - Automatically schedule review sessions
- **Teacher Insights** - Show why AI made each recommendation

### Technical Upgrades

- IRT (Item Response Theory) model for question difficulty
- Retention decay algorithms
- Recommendation engine with explainable logic
- Enhanced analytics with cognitive insights

---

## Phase 3: AI Integration

**Goal:** AI-powered content and tutoring

### AI Features

- **AI Tutor Chat** - Context-aware help for each lesson
- **Theory Simplifier** - AI rewrites explanations based on student level
- **Question Generator** - Create new questions on weak topics
- **Content Quality** - AI flags lessons that confuse students
- **Personalized Pathways** - Dynamic learning paths per student

### Technical Implementation

- OpenAI/Claude API integration
- Context-aware prompt engineering
- Content generation and validation
- Human-in-the-loop approval system
