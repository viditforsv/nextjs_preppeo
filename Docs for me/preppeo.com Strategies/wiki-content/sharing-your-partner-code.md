# Sharing Your Partner Code

Your partner referral code gives students a discount and earns you commission on every purchase.

---

## Your Partner Code

After joining the partner program, you receive a unique code like **COACHING30**. This code:
- Gives students a **10% discount** on mock token purchases
- Earns you **30% commission** on the amount paid

---

## How Students Use Your Code

1. Students go to the Token Store at [preppeo.com/mocks/tokens](https://preppeo.com/mocks/tokens).
2. They select a mock pack.
3. Before checkout, they enter your code in the **Referral Code** field.

![Student entering partner code](placeholder-student-partner-code.png)

4. The discount is applied automatically, and the student sees the reduced price.

![Discount applied at checkout](placeholder-partner-discount.png)

---

## Best Practices for Sharing

- **In class:** Write your code on the board or include it in handouts.
- **WhatsApp/email:** Send a message like: "Use code COACHING30 at preppeo.com/mocks/tokens to get 10% off SAT mocks."
- **On your website:** Add a banner or link with your code.

---

## Important Notes

- The code is **case-insensitive** — COACHING30 and coaching30 both work.
- The discount only applies to **paid purchases**, not free mock claims.
- There is no limit to how many students can use your code.
- If a student uses a different code, you won't receive commission for that purchase.
