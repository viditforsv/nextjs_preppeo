# Using a Referral Code

If you received a referral or partner code, you can apply it to get a discount on mock token purchases.

---

## Where to Enter the Code

### At Checkout (Token Store)

1. Go to the Token Store at [preppeo.com/mocks/tokens](https://preppeo.com/mocks/tokens).
2. Select a mock pack.
3. Before clicking **Buy Now**, enter the referral code in the "Referral Code" field.

![Referral code field at checkout](placeholder-referral-checkout.png)

4. The discount is applied automatically. You'll see the original price, discount amount, and final price.

![Discount applied](placeholder-discount-applied.png)

---

### When Claiming Your Free Mock

1. If a friend shared their referral link, clicking it takes you to the free mock claim page with the code pre-filled.
2. If you have a code but arrived at the page directly, enter it in the "Referral Code" field on the [free mock page](https://preppeo.com/sat-free).

![Referral code on free mock page](placeholder-referral-free-mock.png)

---

## Types of Codes

| Code Type | Who Provides It | What You Get |
|-----------|----------------|--------------|
| Friend referral | Another student | Both get a bonus mock token |
| Partner code | A teacher or institute | 10% discount on purchases |

---

## Troubleshooting

- **"Invalid code"** — Double-check the code for typos. Codes are case-insensitive.
- **Code not applying a discount** — Partner codes only apply to paid purchases, not free mock claims.
- **Already used a friend's code** — Each friend pair can only earn the referral bonus once.
