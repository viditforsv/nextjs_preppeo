# Ashoka Aptitude Test (AAT) — Exam Format for CBT

---

## 1. Exam Overview

The Ashoka Aptitude Test is a **computer-based test** used for undergraduate admissions at Ashoka University. It evaluates reasoning and critical thinking, not school-syllabus knowledge.

| Parameter              | Details                                    |
| :--------------------- | :----------------------------------------- |
| **Mode**               | Computer-Based Test (CBT)                  |
| **Total Questions**    | 40 MCQs + 1 Essay                         |
| **MCQ Duration**       | 90 minutes (single timer, no section lock) |
| **Essay Duration**     | 30 minutes                                 |
| **Sections**           | Problem Solving, Critical Thinking         |
| **Question Order**     | Jumbled across both sections               |
| **Negative Marking**   | None                                       |
| **Calculator**         | Not allowed (rough sheets provided)        |
| **Sectional Cutoffs**  | None                                       |
| **Sectional Time Cap** | None — single 90-min timer for all 40 MCQs |

---

## 2. Section Structure

The MCQ portion is divided into two equal sections. Questions from both sections are **interleaved in random order** during the actual exam.

| Section            | Questions | Focus Areas                                    |
| :----------------- | :-------: | :--------------------------------------------- |
| Problem Solving    |    20     | Numerical reasoning, patterns, logic, data     |
| Critical Thinking  |    20     | Reading passages, argument analysis, inference  |

---

## 3. Problem Solving — Topics & Distribution

Focuses on basic quantitative skills and logical reasoning. No advanced math.

| Topic                                  | Approx. Questions |
| :------------------------------------- | :---------------: |
| Linear Equations                       |         4         |
| Ratio                                  |         2         |
| Percentage / Profit–Loss               |         3         |
| Time, Speed & Distance / Time & Work   |         2         |
| Numbers (Basics)                       |         1         |
| Data Interpretation / Logical Reasoning|         9         |
| **Total**                              |      **20**       |

- Good attempt: 17+ questions
- Good score: 15+ marks
- Difficulty: Moderate — emphasises logic over calculation

---

## 4. Critical Thinking — Topics & Distribution

Tests comprehension and reasoning using short passages and logical scenarios. No subject knowledge required.

| Topic                          | Approx. Questions |
| :----------------------------- | :---------------: |
| Reading Comprehension          |         2         |
| Main Idea                      |         3         |
| Primary Purpose                |         3         |
| Strengthen / Weaken Argument   |        10         |
| Parallel Reasoning             |         2         |
| **Total**                      |      **20**       |

- Good attempt: 16+ questions
- Good score: 14+ marks
- Difficulty: Moderate — accuracy over speed

---

## 5. Marking Scheme

| Component       | Marks |
| :-------------- | :---: |
| Correct Answer  |  +1   |
| Wrong Answer    |   0   |
| Unattempted     |   0   |
| **Total Marks** | **40**|

- Competitive score: **29–30 out of 40**
- No IRT weighting — straightforward 1-mark-per-question scoring.

---

## 6. Essay Component

After the MCQ phase, students enter a separate 30-minute essay phase.

- **Prompts:** 2 options, choose 1.
- **Evaluated on:** Clarity of argument, writing structure, reasoning ability, personal insights.
- **Not evaluated on:** Vocabulary or grammar perfection.

---

## 7. CBT Platform Strategy

Unlike the SAT or GRE, the AAT is a **single-stage, non-adaptive** test.

| Aspect           | Implementation                                            |
| :--------------- | :-------------------------------------------------------- |
| Adaptive routing | None — all students get the same question set             |
| Question order   | Randomised across both sections                           |
| MCQ timer        | Single 90-minute countdown for all 40 questions           |
| Essay timer      | Separate 30-minute phase after MCQ submission             |
| Calculator       | Not needed — no on-screen calculator                      |
| Navigation       | Students can move freely between questions within the MCQ phase |
| Section labels   | Optional — real exam jumbles them, but labelling aids review |
